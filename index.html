How "Cosine" Vectoring Works

The tool doesn't just "look" at pixels; it converts the CT image into a high-dimensional feature vector ($A$). This mathematical signature is compared against centroids of known malignant clusters ($B$) using the standard cosine similarity metric:

Similarity = cos(θ) = (A·B) / (||A|| ||B||)

A score near 1.0 (closer angular proximity) suggests the input vector is nearly identical to the pathological reference pattern.
